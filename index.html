GitHub Pages + WebRTC

Chat direct entre deux pages web.

Ouvre cette page sur deux navigateurs, cree une offre d'un cote, colle la reponse de l'autre, puis discute via un canal WebRTC sans backend applicatif.

Hors ligne Aucun role

1. Initialiser la connexion

Le host cree une offre puis partage un lien d'invitation. L'invite ouvre ce lien et genere une reponse.

  1. Host: clique sur Creer une offre, puis copie le lien d'invitation genere.
  2. Invite: ouvre ce lien. La page detecte l'offre automatiquement et genere une reponse.
  3. Invite: partage le bloc base64 de reponse au host, puis host clique sur Appliquer la reponse.
