GitHub Pages + WebRTC

Chat direct entre deux pages web.

Ouvre cette page sur deux navigateurs, crée une offre d'un côté, colle la réponse de l'autre, puis discute via un canal WebRTC sans backend applicatif.

Hors ligne Aucun role

1. Initialiser la connexion

Choisis ton côté. L'initiateur crée une offre. Le second colle cette offre et génère une réponse.

  1. Côté A: clique sur Creer une offre puis partage le bloc genere.
  2. Côté B: colle l'offre recue, clique sur Accepter une offre, puis partage la reponse.
  3. Côté A: colle la reponse dans le champ distant et applique-la.

3. Signal distant

4. Messages

Le chat s'active des que le canal de donnees est ouvert.
